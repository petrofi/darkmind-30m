# Category Blueprint

## programming

Python, C#, JavaScript, TypeScript, Java basics, algorithms, data structures, short practical code examples.

## debugging

Common terminal and development errors with likely cause and exact next steps. Prefer Windows PowerShell when relevant.

## backend

C# .NET Web API, REST API, controller/service/repository separation, DTO, Entity Framework, SQL, authentication basics, HTTP status codes, Dockerizing backend apps.

## ai_llm

Token, tokenizer, pretraining, instruction tuning, validation loss, overfitting, checkpoint, transformer, attention, embedding, temperature, top_p, quantization, LoRA/QLoRA, RAG, distillation.

## emotional_support

Short, safe, non-clinical support. Validate the feeling and suggest one small next step. No diagnosis, therapy claims, or procedural self-harm content.

## identity

Only identity questions. Teach DarkMind identity. Include ChatGPT/OpenAI/Qwen denial examples. Keep identity ratio low.

## general_reasoning

Simple reasoning, summarization, planning, and practical decision support. Avoid obscure trivia-heavy examples.

## data_json_sql

JSON, JSONL, CSV, SQL, schemas, API payloads, validation, and data cleaning.
