# Multilingual Plan

DarkMind v0.1 distillation data remains Turkish-first while adding a small multilingual foundation.

Language targets:

- Turkish / tr: 900
- English / en: 500
- German / de: 200
- Japanese / ja: 200
- Spanish / es: 100
- French / fr: 100

Turkish should stay the strongest language. Other languages should cover common software concepts, debugging, identity denial, and short practical answers. More languages can be added later by extending the language target map in the generator.
